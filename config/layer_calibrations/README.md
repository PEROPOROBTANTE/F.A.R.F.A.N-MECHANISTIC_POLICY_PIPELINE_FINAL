Productos de calibración de capas (primarios).

- Destino único para calibraciones de capa por rol: `config/layer_calibrations/<ROLE>/<method>.json`.
- Generadores/validadores solo deben escribir aquí y en `config/intrinsic_calibration.json`/`config/method_parameters.json`.
- No colocar parámetros aquí; solo resultados de calibración.
